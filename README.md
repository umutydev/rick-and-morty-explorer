# Rick and Morty Character Finder

This is a React app that fetches and displays characters from the [Rick and Morty API](https://rickandmortyapi.com/). Users can filter characters by name, status, and gender.

## 🚀 Features

- ✅ Character filtering (name, status, gender)
- ✅ Pagination
- ✅ Character detail modal
- ✅ Responsive layout with Tailwind CSS

---

## 📦 Tech Stack

- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)
- [Rick and Morty API](https://rickandmortyapi.com/)
